# guide-biz-ayx

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，内容涵盖业务指南、流程说明及辅助工具页面。所有页面均为静态 HTML，不依赖后端服务，可直接通过浏览器访问。

仓库不针对特定网站或服务，仅作为页面资源的集中存放与版本管理使用。

## 目录说明

```
guide-biz-ayx/
├── pages/          # 存放主要 HTML 页面文件
├── assets/         # 页面依赖的静态资源，如样式表、图片、脚本等
├── docs/           # 相关说明文档或辅助资料
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，按功能或主题命名。
- `assets/`：资源文件按页面或类型组织子目录。
- `docs/`：存放与页面内容相关的说明、参考或变更记录。

## 页面归档说明

- 所有页面均为独立完整的 HTML 文件，可离线或在线直接打开。
- 页面之间无强依赖关系，可单独使用或引用。
- 归档内容保持原始结构和样式，仅进行必要的兼容性调整。
- 页面版本通过 Git 提交记录管理，历史版本可追溯。

## 维护说明

- 欢迎提交 Issue 反馈页面问题或改进建议。
- 如需新增页面，请按照现有目录结构放置文件，并在提交时简要说明用途。
- 不建议对本仓库内容进行大规模重构，保持页面独立性和简洁性。
- 本仓库仅为归档用途，不提供运行环境或服务器配置支持。
